# RadPass Website

Marketing website for RadPass.

## Tech stack
- React
- Vite
- Tailwind
- GitHub Pages

## Deployment

The site is automatically deployed using GitHub Actions.

Workflow:
1. Edit source files in `src/` or `public/`
2. Push to `main`
3. GitHub builds the project (`npm run build`)
4. The `dist` folder is deployed to GitHub Pages

Live site: https://radpass.co.uk
